Для того, чтобы передать данные (Model) из контроллера (из метода-обработчика запроса), в представление (View) необходимо использовать объект класса Model. 
Для того, чтобы его использовать, его необходимо передать в качестве аргуметна в метод-обработчик запроса.

Допустим нам необходимо передать строку, содержащую имя в представление:

@RequestMapping(value="/example", method=RequestMethod.GET)
public String viewName(Model model){
		String name = "Leha";
		model.addAttribute("name", name);
		return "hello";
}

--------------------------

Для того, чтобы вывести значение переменной во View (hello.jsp) необходимо использовать конструкцию ${name}.
Например выведем в виде заголовка:

<h1>
	${name}
</h1>

--------------------------

Из JAVA кода внутри JSP доступ к переменной модели (в нашем случае name) можно получить следующим образом:

<%
	String name = (String) request.getAttribute("name");
%>

И далее такую переменную можно вывести в JSP следующим образом:

<h1>
	<%= name %>
</h1>

--------------------------

В обоих случаях в результате получим HTML:

<h1>
	Leha
</h1>
