Перечисления (enumerations)

1) Перечисление - это список именованных констант, которые называют константы перечислимого типа. (см. пример стр. 319)

2) Перечисления в Java реализуются в виде класса, т.е. могут иметь конструкторы, поля (переменные экземпляра), методы.

3) Нельзя напрямую создать экземпляр перечисления (т.е. использовать оператор new).

4) КАЖДАЯ КОНСТАНТА ПЕРЕЧИСЛЕНИЯ ЯВЛЯЕТСЯ ОБЪЕКТОМ (ЭКЗЕМПЛЯРОМ) ЭТОГО КЛАССА (ПЕРЕЧИСЛЕНИЯ). (см. пример стр. 321)
Т.е. в процессе создания константы перечислимого типа вызывается конструктор перечислимого типа.
Также в этом созданном объекте (в созданной константе) доступны все объявленные в перечислимом типе поля и методы. 

5) Перечисление не может быть суперклассом.

6) Перечисление не может наследоваться от другого суперкласса.
