Рекурсивная функция - это ф-я, которая вызывает саму себя до достижения условия остановки.

Для посторения рекурсивной функции необходимо определить:
1) условие остановки (так называемый "базовый случай")
2) условие продожения (так называемый "шаг рекурсии")


Рассмотрим создание рекурсивной ф-ии на примере вычисления факториала числа, т.е. произведения числа на все его предшествующие числа.
Например, факториал 5 прдеставляется как 5 * 4 * 3 * 2 * 1 и равен 120
Или в общем виде факториал можно представить в виде: n * (n-1) * (n-2) * ... * (n-(n-1)) 

------------------------

package main;


public class Main { 
    
    public static void main(String[] args) {         
        System.out.println("Факториал 5 = "+factorial(5));        
    } 
    
    // Объявляем рекурсивную ф-ю для вычисления факторила
    public static int factorial(int i){
        // Объявляем условие остановки
        if (i == 1) return 1;
        // Объявляем шаг рекурсии
        return i * factorial(i-1);
    }
    
}
