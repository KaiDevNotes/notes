1) Создаем Web-проект: 

# Java Web -> Web-приложение. 
# В данном проекте в папке Веб-страницы создаем JSP-страницу index.jsp. Код:

#------------------------------- index.jsp ------------------------------------------------------------

<%@page contentType="text/html" pageEncoding="UTF-8"%>
<!DOCTYPE html>
<html>
    <head>
        <meta http-equiv="Content-Type" content="text/html; charset=UTF-8">
        <title>Local host info</title>
    </head>
    <script language="javascript">
        
        function writeStr(str){
            var output = document.getElementById("output");
            output.innerHTML = str;
        }
        
    </script>
    <body>
        <h1>LOCAL HOST INFO:</h1>
        <h3 id="output"></h1>
        <p>
            <input type="button" value="GET HOST NAME" onclick="document.applets[0].getHostName()">
            &nbsp;&nbsp;&nbsp;
            <input type="button" value="GET HOST IP" onclick="document.applets[0].getHostIP()">
        </p>
        <br/>
        <applet code="test.applet.one.Applet1" archive="TestApplet.jar" width="600" height="480"/>        
    </body>
</html>

#------------------------------------------------------------------------------------------------------
# В коде страницы есть:
- h3#output для вывода результатов нажатия клавиш  
- клавиша GET HOST NAME - вызывает метод applet'a getHostName()
- клавиша GET HOST IP - вызывает метод applet'a getHostIP()
- и сам applet test.applet.one.Applet1 из библиотеки TestApplet.jar



2) Необходимо создать и протестировать applet, в нашем случае это test.applet.one.Applet1 
(для этого необходимо создать отдельное Java-приложение, собрать и затем только интегрировать в web-проект).
Код applet'a:
  
#------------------------------- Applet1.java ---------------------------------------------------------

package test.applet.one;

import java.applet.Applet;
import java.awt.Graphics;
import java.io.*;
import java.net.*;
import netscape.javascript.*;

public class Applet1 extends Applet {

    private String result = "Default Address";
    
    @Override
    public void paint(Graphics g) {
        g.drawString("Applet successfully loaded", 50, 25);
        JSObject win = JSObject.getWindow(this);
        String argums[] = new String[1];
        argums[0] = "Click button to get result...";
        win.call("writeStr", argums);
    }
    
    public void getHostName(){
        try {           
           InetAddress address = InetAddress.getByName(null);
           result = "Host Name: "+address.getHostName();
        }
        catch (IOException e){}
        JSObject win = JSObject.getWindow(this);
        String argums[] = new String[1];
        argums[0] = result;
        win.call("writeStr", argums);
    }
    
    public void getHostIP(){
        try {           
           InetAddress address = InetAddress.getByName(null);
           result = "Host IP: "+address.getHostAddress();
        }
        catch (IOException e){}
        JSObject win = JSObject.getWindow(this);
        String argums[] = new String[1];
        argums[0] = result;
        win.call("writeStr", argums);
    }
}

#------------------------------------------------------------------------------------------------------

#--------------------------------------------- ОПИСАНИЕ РАБОТЫ ПРИМЕРА ----------------------------------------  

1) При загрузке станицы вызывается метод paint() applet'a, кот. выводит (рисует) в тело applet'a 
надпись "Applet successfully loaded" и вызывает javascript-метод writeStr(), кот. выводит в h3#output
переданную ему строку "Click button to get result..." 

2) При нажатии на клавишу "GET HOST NAME" вызывается метод getHostName() applet'a, который по средствам класса
InetAddress получает имя host'a, на котором была загружена данная страница и на котором запустился данный applet.
Далее метод getHostName() вызывает javascript-метод writeStr(), кот. выводит в h3#output
переданную ему строку имени host'a.

3) При нажатии на клавишу "GET HOST IP" вызывается метод getHostIP() applet'a, который по средствам класса
InetAddress получает ip-адрес host'a, на котором была загружена данная страница и на котором запустился данный applet.
Далее метод getHostIP() вызывает javascript-метод writeStr(), кот. выводит в h3#output
переданную ему строку ip-адреса host'a.

ЗАМЕЧАНИЕ:

В JDK8 есть bug связанный с методом getWindow объекта JSObject. Его описание:
https://netbeans.org/bugzilla/show_bug.cgi?id=243293

Решение:

Необходимо заменить (добавить) следующий кусок кода в nbproject/project.properties

endorsed.classpath=\
    ${java.home}/lib/javaws.jar:\
    ${java.home}/lib/plugin.jar
