Разница между GET и POST. 
-------------------------

http://stackoverflow.com/questions/3477333/what-is-the-difference-between-post-and-get


Способ передачи информации:
- GET: в адресной строке, как часть URL
- POST: в теле запроса

Кеширование:
- GET: кешируется
- POST: не кешируется

Предельный размер запроса:

- GET: определяются как настройками сервера, так и клиентом (браузером).
https://technomanor.wordpress.com/2012/04/03/maximum-url-size/
Для Tomcat предельный размер GET запроса составляет 8Kb by default и 64Kb, если добавить 
maxHttpHeaderSize="65536" к HTTP 8080 connector в conf/server.xml
<Connector port="8080" maxHttpHeaderSize="65536" ... />
http://serverfault.com/questions/56691/whats-the-maximum-url-length-in-tomcat

- POST: определяются настройками сервера.
Для Tomcat предельный размер POST запроса составляет 2Mb by default и неограничен, если 
добавить maxPostSize="0" к HTTP 8080 connector в conf/server.xml
<Connector port="8080" maxPostSize="0" ... />
https://www.enterprise-architecture.org/documentation/doc-administration/145-post-size-limit
