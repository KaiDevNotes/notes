Для реализации SSH клиента на Java есть 2 варианта:
1) Непосредственно использовать библиотеку JSch.
2) Или использовать оболочку для JSch под названием JCABI (описыается нижу)



Для использования библиотеки JCABI в CLASSPATH необходимо добавить следующие библиотеки:

jcabi-ssh-1.5.2.jar
jsch-0.1.42.jar
commons-lang3-3.0.1.jar
commons-io-2.4.jar
logback-core-0.9.30.jar
logback-classic-0.9.30.jar
slf4j-api-1.6.1.jar
slf4j-log4j12-1.6.4.jar
jcabi-log-0.17.1.jar



Пример выполнения SSH запроса:
-------------------------------

package main;

import com.jcabi.ssh.SSHByPassword;
import com.jcabi.ssh.Shell;
import java.io.IOException;
import java.net.UnknownHostException;


public class Main {
    
    public static void main(String[] args) throws UnknownHostException, IOException, InterruptedException {
        
        SSHByPassword ssh = new SSHByPassword(host, port, user, password);  
        String r = new Shell.Plain(ssh).exec("echo 'Hello world!'");
        System.out.println(r);   
        
    }    
}


Конструктору экземпляра SSHByPassword необходимо передать реальные host, port, user, password.


В результате в консоль получаем результат выполнения команды echo 'Hello world!', которое будет просто:

Hello world!
