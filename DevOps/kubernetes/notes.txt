

Kubectl commands cheatsheet from official documentation:
https://kubernetes.io/docs/reference/kubectl/cheatsheet/


-----------------------------


Start Minikube: 

Open PowerShell as administrator and execute next command:
|
|
$ minikube start --vm-driver=virtualbox
|
|


-----------------------------


To find Node IP when using Minikube we should execute next command
|
|
$ minikube ip
|
|
